🎤 Microphone Permission (Voice Name Entry)

This game asks your child's name using her voice. The very first time:

  1. A pop-up will say "Allow microphone?"
  2. Tap Allow
  3. That's it — it never asks again.

If the mic doesn't catch the name after 3 tries, Baby Shark will give her a fun princess name automatically — no worries!

📱 Playing on iPhone or iPad

Important: Voice name entry requires a secure web link (https://...). It will NOT work from a local file on iPhone/iPad. Use the public link for this game — ask Corey for the URL.

Once you have the link, open it in Safari on iPhone or iPad for the best experience.

🏠 Add to iPhone / iPad Home Screen (makes it feel like an app!)

  1. Open the game link in Safari
  2. Tap the Share button at the bottom of Safari (the box with an arrow pointing up ↑)
  3. Scroll down in the menu and tap "Add to Home Screen"
  4. Tap "Add" in the top-right corner
  5. Done! The game icon now appears on the Home Screen — tap it to open fullscreen, just like an app

💡 Tips for best play

🔊 Turn volume up before starting
🤫 Play in a quiet room for best voice recognition
🖐️ All interactions are tap-only — no reading or typing needed
👧 Designed for ages 4–6
